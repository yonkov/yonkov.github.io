4 Awesome Things You Can Do with Hugo

November 7, 2017

Hugo is an awesome static website generator, which you can use to create beautiful portfolio websites and blogs. So, what makes Hugo so great?

Speed

Hugo is absurdly fast! The Developers claim that a webpage can render just for 1 milisecond!

Functionalities

1. Create a list page or a post using default templates.

There are two types of templates you can use in Hugo. These are the single page templates and list page templates. There can also be a different page template for the home page. You can also create your own custom page templates.

2. Shortcodes

Hugo is written in Go language and comes with pre-defined shortcodes. You can use a shortcode instead of custom HTML to embed an youtube video. Just like that:


You can use a nice shortcode to highlight code:

<section id="main">
  <div>
   <h1 id="title">{{ .Title }}</h1>
    {{ range .Data.Pages }}
        {{ .Render "summary"}}
    {{ end }}
  </div>
</section>


You can also use a shortcode to add an image. You can even create your custom shortcodes!

3. HTML and JS inside post content

You can put html or javascript whereever you want! No need for plugins or extra customization.

Disqus

Hugo ships with Disqus template. This is a comment system for static websites. In this way, you can easily include comments in your Hugo blog.
