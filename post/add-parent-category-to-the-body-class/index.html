How To Easily Add Parent Category to the WordPress body class

September 24, 2019
Category: WordPress

I had a client that wanted to customize category archives in such a way that they had different content and different layout for the child categories that belong to a specific parent category. For this, I needed a class name, so i can display it on the frontend and customize each child category’s archive page accordingly. A. In this tutorial, I will show you how to quickly assign the parent category id as a body class name to any WordPress child category archive. Tested and works on my website.

Add Parent Category Id to Body Class

Here is the code that I used to display the parent category class name on the body tag of each child category. Add this on top of your child’s theme header.php:

<?php
    $this_category = get_category($cat);
    if ($this_category->category_parent == 0) {
    $this_category->category_parent = $cat;
    } else {
    $parent_category = get_category($this_category->category_parent);
    $ParentCatId = $parent_category->cat_ID;
    }
    $childcategories = array();
    $catx = $ParentCatId;
    $cats = get_categories('parent='.$catx);
    foreach($cats as $cat) {
    $childcategories[] = $cat->term_id; } ;
    if( is_category( $childcategories ) ) {
    $class = 'parent-category-'.$ParentCatId;
    }
?>

Now, pass the $class variable as an argument to the body_class function in header.php. Locate the body tag and update it to look like this:

<body <?php body_class($class); ?>>

Now, when you open a child category’s archive page and inspect the page with the developer tools (F11), you should be able to see a body classname for the child categories that come from different parent categories. For example, if you want to add styles to the child category that has a parent category with id==2, you can use the following css code: .parent-category-2{ /* Add your css code here */ }
